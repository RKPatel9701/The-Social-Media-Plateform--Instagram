# Instagram Clone Database
### Database Management System (DBMS) Project

## Overview

This project presents the design and implementation of a relational database for an Instagram-like social media platform. It models the core features of a modern social networking application using PostgreSQL and SQL while following database design principles such as normalization, referential integrity, and relationship modeling.

The system supports essential Instagram functionalities, including user management, posts, reels, stories, likes, comments, followers, messaging, calls, highlights, and close friends.
